# Locus Resources

This folder contains application resources:

- `icon.ico` - Windows application icon (add your custom icon here)
- `icon.png` - PNG version of the icon for other platforms

## Icon Requirements

For Windows builds, you need an `.ico` file with the following sizes:
- 16x16
- 32x32
- 48x48
- 64x64
- 128x128
- 256x256

You can use tools like:
- [ICO Convert](https://icoconvert.com/)
- [ConvertICO](https://www.convertico.com/)
- ImageMagick CLI
